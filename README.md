# Sistema de Gestión de Vehículos - RentaFácil

## Autor
Nicolás Rodriguez

## Descripción
Sistema desarrollado con Spring Boot y PostgreSQL para la gestión de vehículos de alquiler.

## Tecnologías utilizadas
- Java 21
- Spring Boot
- Spring Data JPA
- PostgreSQL
- Maven
- HTML
- CSS
- JavaScript

## Funcionalidades
- Crear vehículo
- Listar vehículos
- Buscar vehículo por ID
- Actualizar vehículo
- Eliminar vehículo
- Consulta por categoría
- Consulta por rango de precio
- Consulta por modelo
